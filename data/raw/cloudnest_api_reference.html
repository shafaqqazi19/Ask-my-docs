CloudNest API Reference

Base URL: https://api.cloudnest.example/v1

Authentication

All requests must include a bearer token in the Authorization header, obtained from Settings > API Keys. Tokens do not expire but can be revoked at any time from the same page.

Rate Limits

Rate limits are enforced per workspace, not per token.

PlanRequests per Minute
Free30
Pro120
Enterprise600

Exceeding the limit returns HTTP 429 with a Retry-After header indicating seconds until the limit resets.

Endpoints

List Files

GET /files — returns up to 100 files per page. Use the cursor query parameter for pagination.

Upload File

POST /files — accepts multipart/form-data. Maximum file size is 25 MB on the Free plan and 5 GB on Pro and Enterprise plans.

Delete File

DELETE /files/{id} — soft-deletes the file; it is recoverable from the Trash for 30 days before permanent deletion.

Get Current User

GET /users/me — returns the authenticated user's profile and workspace role.
